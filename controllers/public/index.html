POS Simulator

Using the app, baristas can effectively: learn how to take actual customer orders, become familiar with the layout of the POS system, and become more comfortable asking questions to customers so they can make the moment right with every order. This project takes in many parameters to wholly simulate on-the-job scenarios, so partners feel comfortable before their first day on the floor.
